## 🧮 Atividade 1 - Teste de Mesa
**Cálculo de Médias - Análise Detalhada**

Foram realizadas 10 simulações distintas utilizando a fórmula de cálculo de média aritmética, com diferentes conjuntos de notas.

**📎 Documentação:**
- [Relatório Completo do Teste de Mesa](Documentos/GitHub/atividade-java-uml/Atividade_8/docs/pdf/TestedeMesa)

**🔍 Metodologia:**
- Cada cenário foi calculado manualmente
- Validação posterior através do programa Java
- Análise comparativa dos resultados

## 🖥️ Atividade Extra - Interfaces Gráficas
**Desenvolvimento com SceneBuilder**

**Características das Interfaces:**
- Campos para entrada de todos os atributos
- Botão "Salvar" para persistência dos dados
- Geração de arquivos CSV com os objetos criados
- Layout intuitivo e responsivo

**Tecnologias Utilizadas:**
- JavaFX para interface gráfica
- SceneBuilder para design visual
- Persistência em formato CSV
